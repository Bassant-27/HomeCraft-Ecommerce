# 🏠 HomeCraft - Furniture E-Commerce Website

A full-stack furniture e-commerce web app built with Flask and Python.

> 👥 Built collaboratively by a 4-person team

## Tech Stack
Flask | Python | SQLite | MVC | Pytest | Docker

## Features
- Product browsing & filtering
- Shopping cart & wishlist
- Admin dashboard
- Automated testing with Pytest

## How to Run
git clone https://github.com/Bassant-27/HomeCraft-Ecommerce.git
cd HomeCraft-Ecommerce
pip install -r requirements.txt
python main.py
